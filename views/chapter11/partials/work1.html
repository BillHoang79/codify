Goal: Create a 7 question quiz app with answers stored in an array of objects

The user should be able to answers the questions in the quiz and have the results displayed to them on the screen.

There will be many steps to the project, so take it one step at a time.

This will be one of your codepen/portfolio projects, so make sure to add in cool functionality and style it in a unique way. Here are some functionality ideas: try randomizing questions, change the color of the questions based on if they were answered correctly or not, add in a congratulations popup if every answer is right.
