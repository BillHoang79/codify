Write an app that searches by hashtag and collects submissions from Instagram and then displays the returned data to the screen with newly created JavaScript elements.

Instagram hashtags: https://www.instagram.com/developer/endpoints/tags/#get_tags

You will need an Instagram Access Token, found here: http://instagram.pixelunion.net/ (if you don't have instagram, you will need to sign up real quick)

In your JS you can utilize $.ajax() or $http.jsonp() to call the data into your application.

Finally Display it to the screen for the user to see, consider wrapping each post in an a tag, you can link to their instagram account!

Add any additional functionality you'd like to see: search by date, pagination for additional pages, fun animations.
